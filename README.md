# 🏪 Sistema de Gestión de Tienda

## 👨🏻‍💻 Integrante A – Guillermo

### HU-01 – Registrar Producto
Como administrador  
Quiero registrar nuevos productos en el sistema  
Para mantener actualizado el catálogo de la tienda.

### HU-02 – Actualizar Producto
Como administrador  
Quiero modificar el precio y stock de un producto existente  
Para reflejar cambios reales en el inventario.

## 👨🏻‍💻 Integrante B - Miguel 

### HU-03 – Registrar Cliente
Como vendedor  
Quiero registrar nuevos clientes  
Para asociar correctamente las ventas realizadas.

### HU-04 – Registrar Venta
Como vendedor  
Quiero registrar una venta seleccionando cliente y productos  
Para guardar el historial de compras.

### HU-05 – Validación y Descuento Automático de Stock
Como sistema  
Quiero verificar que un producto tenga stock disponible antes de venderlo  
Para evitar ventas de productos agotados y mantener el inventario actualizado.
